# Backlink Outreach Pack 3 - Different Channels Only

Reference date: March 14, 2026

Purpose:

- This pack is intentionally different from the university / municipality / library / coach-network outreach used in the first waves.
- Use it to diversify backlink acquisition into expat centers, event listings, partner programs, and international-career ecosystems.

How to use this pack:

1. Do not send these in the same week as another large outreach wave.
2. Prioritize the `send-now` targets first.
3. Use the `calendar-watch` targets to build relationships or secure the next event cycle.
4. Pitch one primary URL only, then offer a small follow-up pack if they reply.
