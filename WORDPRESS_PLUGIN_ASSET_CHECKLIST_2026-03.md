## Visual Direction

Keep the listing visuals aligned with WerkCV:

- background: warm off-white or clean white
- accent: yellow and teal, matching the current site
- strong black borders, clear shapes
- avoid cluttered marketing graphics
- show the plugin as practical, Dutch, and utility-first

## Banner Copy Direction

Suggested headline:

- `Nederlandse salaris-tools voor WordPress`

Suggested subline:

- `Netto-bruto, vakantiegeld en minimumloon via hosted embeds van WerkCV.nl`

## Icon Direction

Simple options:

- calculator glyph + `NL`
- calculator glyph + `€`
- mini WerkCV badge + calculator

Keep it legible at small size.

## Submission-Ready Readme Checks

Before submitting:

- confirm `Tested up to` matches the WordPress version you tested locally
- confirm screenshots listed in `readme.txt` exactly match the uploaded files
- confirm there is no claim of forced backlinking
- confirm support/privacy URLs point to:
  - `/wordpress/salaris-tools-plugin`
  - `/wordpress/salaris-tools-plugin/privacy`
  - `/wordpress/salaris-tools-plugin/installatie`

## Final Pre-Submit Checklist

- shortcode path works
- Gutenberg block path works
- plugin settings work
- screenshots exported
- icon and banner exported
- `readme.txt` finalized
- plugin zip generated
- WordPress.org submission account ready
