# Chrome Web Store Draft

## Name

WerkCV Vacature Keyword Highlighter

## Short description

Highlight Dutch and English job-page keywords and surface the terms worth reusing in your CV.

## Detailed description

WerkCV Vacature Keyword Highlighter scans the active Dutch or English job page, highlights CV-relevant wording directly on the page, and groups the strongest terms inside the popup.

Use it to spot:

- hard skills such as WordPress, SEO, GA4, Google Analytics, HubSpot, or Excel
- soft skills such as analytical, collaborative, klantgericht, or communicatief sterk
- qualifications such as HBO, bachelor, Dutch, and English
- repeated themes worth reusing carefully in a CV or motivation letter

The extension is intentionally narrow:

- it works on the current tab only
- it runs locally in the browser
- it does not require an account
- it does not run background scraping across your tabs

Best fit:

- Dutch vacature pages
- English job pages
- users tailoring a CV to a live vacancy before applying

## Single purpose

This extension scans the active Dutch or English job page, highlights CV-relevant keywords on the page, and shows a grouped keyword summary inside the popup.

## Permissions justification

- `activeTab`: required to scan only the page the user explicitly chooses
- `scripting`: required to inject the local highlighter into that page

## Privacy positioning

- no account required
- no remote code
- no background scraping
- page analysis runs in the browser on the active tab
